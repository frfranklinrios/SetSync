Primeiros passos
  1. Faça login (ou registre uma conta).
  2. Crie uma banda em Bandas.
  3. Cadastre cifras dentro da banda (manual ou via Cifra → SetSync).
  4. Crie um setlist e entre no Modo Tocar para executar ao vivo.
Dica: no palco, use o Modo Tocar em tela cheia e ative 2 colunas para evitar rolagem em cifras longas.
Bandas e membros

Organização

  • Use uma banda por projeto (ex.: “Trio Acústico”, “Banda de baile”).
  • Centralize as cifras no SetSync para evitar versões diferentes em grupos/WhatsApp.
Cifras

Cadastro

  • Título/Artista: usados em buscas e no Modo Tocar.
  • Tom: referência para transposição (ex.: F#m, Bb).
  • Conteúdo: pode conter acordes no formato [Am] misturados com letra.

Tablaturas

  • Quando a cifra contém blocos de tablatura (linhas com e|, B|, etc.), o SetSync prioriza a renderização em texto para preservar o desenho.
Transposição
  • Você escolhe o tom alvo e o SetSync recalcula os acordes na hora (sem alterar o texto original salvo).
  • Use a transposição para adaptar a música à voz do vocalista sem reescrever a cifra.
Setlists
  • O setlist define a ordem de execução.
  • No Modo Tocar, você navega com ← → ou por gesto (swipe).
Modo Tocar

Atalhos

  • P: entrar no modo tocar
  • Esc: sair
  • ← / →: música anterior / próxima
  • Espaço: auto-scroll play/pause
  • + / -: tamanho da fonte
  • C: alternar 2 colunas
  • S: abrir/fechar setlist
Importar cifra (Cifra → SetSync)

Em Adicionar ou Editar cifra, clique em Cifra → SetSync. O SetSync abre um iframe com a ferramenta integrada (/cifras/import/tool) e preenche o formulário ao final.

Modos (sem login no YouTube)

  • Enviar áudio — você envia MP3/WAV da faixa; o servidor analisa o áudio e monta a grade (recomendado no Contabo).
  • Somente cifra — importa texto e acordes da página; grade pelos acordes (sem BPM automático).
  • YouTube no servidor — só aparece se CIFRAS_YOUTUBE_NO_SERVER=0; exige cookies no servidor e costuma falhar em VPS.

Requisitos

  • ffmpeg no servidor (para converter áudio enviado).
  • Playwright + Chromium (opcional, para vídeo do player do Cifra Club).
PWA / Instalação
  • Android / desktop: use o botão Instalar no topo quando o navegador oferecer.
  • iPhone / iPad: abra no Safari, toque Instalar e siga o guia (Compartilhar → Adicionar à Tela de Início). No iOS o Chrome não instala como app completo.
  • O app usa service worker para ícones, página de offline e cache das páginas que você já visitou.
Problemas comuns

“Not Found” ao abrir o modal

  • Faça um hard refresh (Ctrl+Shift+R) para atualizar o cache do PWA.
  • Verifique se a rota /cifras/import/tool está respondendo.

Importador falha com erro de ffmpeg

  • Instale o ffmpeg no sistema e garanta que o comando ffmpeg funciona no terminal.

Playwright/Chromium não encontrado

  • Execute playwright install chromium no ambiente do projeto.
