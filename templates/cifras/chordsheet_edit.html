Meta caracteres (sintaxe)
  • - texto — rótulo; # — comentário; + — nova página
  • = A — seção; A_B — acordes no mesmo compasso; * — vazio
  • (A B C D)x2 — repetição; 1. 2. — voltas
  • % %% — simile; %1 — repete compasso anterior
